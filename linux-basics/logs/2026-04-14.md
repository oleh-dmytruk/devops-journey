VLAN

- VLAN - virtual network that segments physical network into logical parts

- Max 4094 VLANs (12-bit VID field)

- Access port - connects end devices, works with untagged traffic

- Trunk port - connects switches, carries tagged traffic from multiple VLANs

- IEEE 802.1Q (Dot1q) - standard for VLAN tagging in Ethernet frames
