Networking - Day 1-2

ARP
ARP - protocol that determines MAC address based on IP address.

TCP vs UDP
UDP - fast protocol, transfers packets without guarantees, 
only checks checksum at the end.
TCP - reliable protocol, slower, confirms delivery of each packet.

OSI Model
7 layers: Physical → Data Link → Network → Transport → 
Session → Presentation → Application
MAC address - Data Link layer (L2)
IP address - Network layer (L3)

TCP 3-way Handshake
1. Client → SYN → "want to connect"
2. Server → SYN-ACK → "ready"
3. Client → ACK → "connection established"

Default Gateway
Main router that connects local network to the internet.

Broadcast
Last address in a subnet.
Formula: next network - 1
Example /27, network 192.167.43.32: 32+32=64, broadcast=63
